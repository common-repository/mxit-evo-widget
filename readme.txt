Plugin that displays the MXit EVO Webchat widget in a user's Wordpress blog.

== Description ==

The MXit EVO Webchat adds functionality for anyone on your page to access the MXit instant messaging network, in order to view their online contacts and chat with them about the page.

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload `evo_plugin.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Go to the 'MXit Evo' page under the Settings link to configure the plugin.

== Frequently Asked Questions ==

= Can I access my normal MXit account with this widget? =

Yes, the web widget uses the exact same user base as the normal MXit application on your phone! So you can sign in to your normal account, using your phone number, and see and chat to all of your contacts.
